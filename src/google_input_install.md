# Install Google-PinYin

### On Ubuntu, Google-PinYin based on Fcitx. So install Fcitx firstly.

```bash
sudo apt install fcitx
```

### Configure Fcitx

```bash
im-config
```

### Install Google-PinYin

```bash
sudo apt install fcitx-googlepinyin
```

### Running Fcitx

```bash
fcitx-config-gtk3
```
